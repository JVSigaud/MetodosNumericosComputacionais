Instalação bibliotecas

Para instalar bibliotecas, procure por cmd ou Prompt de comando na aba de navegação do Windows. Em seguida digite: pip install nome da biblioteca.
Dentre as bibliotecas que serão utilizadas temos:

  • Numpy

  • Matplotlib

  • Sympy

  • Scipy

  • Jupyter Notebook
